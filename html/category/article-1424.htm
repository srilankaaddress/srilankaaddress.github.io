通过+（字符串串联）运算符进行字符串串联。
  • 首页
  • 文章分类

可以使用Java的“ +”运算符来连接字符串。

示例

public class Test {    public static void main(String args[]){       String st1 = "Hello";       String st2 = "How";       String st3 = "You";       String res = st1+st2+st3;       System.out.println(res);    } }
