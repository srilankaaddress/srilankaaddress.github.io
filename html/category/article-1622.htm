Anaconda 创建django不同版本开发环境

django开发会用 virtualenv 来管理多个开发环境，但是因为我电脑上同时用tensorflow做深度学习，已经安装了Anaconda，所以尝试用conda 管理环境，实践证明是可行的。所以记录一下
分为三个步骤

  • 选择python版本创建环境
  • 在环境内安装django
  • 测试是否正常

选择python版本创建环境

conda create -n name python=3

在环境内安装django

source activate sbgl#激活环境

注意点：不要用pip安装Django，测试发现无法导入django

conda install Django==1.11.8 #安装django

测试是否正常

再Python环境下：

>>> import django >>> django.VERSION (1, 11, 8, 'final', 0) >>> exit()

测试成功，完工
